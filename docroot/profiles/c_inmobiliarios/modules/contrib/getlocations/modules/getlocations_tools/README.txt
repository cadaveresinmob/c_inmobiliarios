Getlocations Tools provides utilities to help in migration and standardisation of Getlocations installs.
You can export and import the configuration settings of Getlocations and submodules Getlocations Fields, Getlocations Search and Getlocations Leaflet.

To export, click on the export tab, select the module you want to export and submit, then you can use copy and paste to save the displayed settings to a text file.

To import, click on the import tab, select the module you want to import and submit, then you can paste the previously saved configuration into the textarea box and save it.
